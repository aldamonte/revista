# revista
Curadoria de conteúdo diário pessoal

Diariamente as 07:00 deve ser disparada a aplicação 

Ela acessa o claude com instruções para gerar uma pagina html com as 10  principais noticias do dia respeitando a curadoria para executivos de tecnologia.

Na sequencia é disparado o link para um bot do telegram com  página que fia no github pages
